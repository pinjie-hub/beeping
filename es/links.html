Sitios de interés

Esta página se puede leer en los siguiente idiomas:

Inglés | Español

GitHub

Si te encuentras algún problema al implementar el protocolo, o bien algún error, o se te ocurre alguna mejora puedes abrir una issue en nuestro Repositorio de Github.

Stack Overflow

Este es el mejor sitio para hacer las preguntas técnicas que tengas sobre el proyecto. Desde nuestra página de Stack Overflow podrás preguntar y responder a todas las cuestiones técnicas que necesites.

Cuando escribas en Stack Overflow acuérdate de usar el tag - beeping -.

Gitter

Desde Gitter hablamos entre nosotros sobre distintos temas relacionados con cada uno de los repositorios.

Aquí os dejamos el acceso a Gitter.

Website

Si queréis tener más información sobre el proyecto podéis entrar en nuestra página web oficial.
